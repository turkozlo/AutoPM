# AutoPM - Minimal Process Data Agent

AutoPM — это легковесный агент для быстрой первичной загрузки и оценки структуры данных бизнес-процессов.

## 🚀 Возможности
*   **Загрузка данных**: Поддержка CSV и Excel файлов.
*   **Автоматическое форматирование**: Агент (через LLM) анализирует данные и автоматически конвертирует числа и даты в правильный формат.
*   **Базовый отчет**: Генерирует простой отчет (`reports/report.md`) с количеством строк, столбцами и их названиями.
*   **Приватный Чат**: Интерфейс для вопросов о **структуре** данных. Агент знает только метаданные (названия колонок) и намеренно отказывается от выполнения кода или глубокой аналитики, обеспечивая безопасность и простоту.

## 🚦 Быстрый старт

### 1. Установка
```bash
git clone <repository_url>
cd AutoPM
pip install -r requirements.txt
```

### 2. Конфигурация
В корне проекта находится файл `config.yaml`. Отредактируйте его, указав ваши настройки:
```yaml
# Пример config.yaml
provider: "mistral" # или "local"

mistral:
  api_key: "ВАШ_API_КЛЮЧ" 
  model: "mistral-small-latest"

local:
  base_url: "http://localhost:11434/v1"
  model: "llama3.2"
  api_key: "ollama"
```

### 3. Запуск
```bash
python pm_agent/main.py --file "data.csv"
```
Или просто запустите `python pm_agent/main.py` и следуйте инструкциям.

После обработки данных вы попадете в чат, где сможете спросить:
*   *"Какие есть колонки?"*
*   *"Сколько всего записей?"*

---
**Примечание**: Это минимальная версия агента без функций Code Interpreter и сложной аналитики.
