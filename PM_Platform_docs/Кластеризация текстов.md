# Кластеризация текстов

Платформа позволяет автоматически группировать текстовые поля (например, комментарии или описания ошибок) с использованием алгоритмов машинного обучения.

## Выполнение кластеризации
1. Перейдите в раздел "Аналитика" -> "Кластеризация".
2. Выберите колонку с текстовыми данными.
3. Настройте количество кластеров или выберите автоматический режим (алгоритм HDBSCAN).
4. Нажмите "Рассчитать".

## Проверка результатов
Результаты отображаются в виде пузырьковой диаграммы. Кликнув по кластеру, вы увидите наиболее характерные слова и примеры записей, попавших в эту группу.

## Сохранение результатов
Кластеры можно сохранить как новую колонку в наборе данных для дальнейшего использования в фильтрах или при построении графа процесса.
