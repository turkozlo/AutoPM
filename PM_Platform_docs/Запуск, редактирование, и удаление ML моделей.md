# Запуск, редактирование, и удаление ML моделей

В этом руководстве описывается процесс работы с ML моделями в нашей Платформе Process Mining.

## Запуск модели
Для запуска модели перейдите в раздел "Аналитика" -> "ML Модели". Выберите нужную модель из списка и нажмите кнопку "Запустить". Укажите параметры обучения и выберите целевой лог событий.

## Редактирование модели
Редактирование доступно только для моделей в статусе "Черновик" или "Остановлена". Вы можете изменить архитектуру нейронной сети или гиперпараметры в окне настроек.

## Удаление модели
Для удаления модели нажмите на иконку корзины рядом с названием. Внимание: это действие необратимо, все данные обучения будут удалены.
